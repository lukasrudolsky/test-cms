# Style Analysis

Use references to extract visual DNA, not to copy shapes.

For each reference, inspect:

- geometric principle;
- simplicity level;
- stroke or mass weight;
- corner radius and edge sharpness;
- negative space;
- symmetry and asymmetry;
- proportions and optical balance;
- symbol construction method;
- symbol and wordmark relationship;
- typographic character;
- color system;
- abstraction level;
- brand impression;
- readability at 16-24 px.

Then write `docs/brand/reference-analysis.md` with a shared style DNA. Prefer concrete language such as "single continuous ribbon with 2:1 stroke-to-gap ratio" over vague labels.
