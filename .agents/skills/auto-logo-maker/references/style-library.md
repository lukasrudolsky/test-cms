# Style Library

Use this reference for `LEARN_STYLE` mode.

## Outputs

Create or update:

- `style-profile.json`
- `style-profile.md`
- `generation-rules.md`
- `avoid-patterns.md`
- `design-system.md`
- `representative-index.json`
- `training-report.md`
- `changelog.md`

Select at most 10-12 representative images. Prefer diversity across symbol types, geometry, abstraction, color, and presentation.

## Analyze

Extract:

- simplification of shapes;
- negative-space behavior;
- geometric connections;
- corner radius and edge sharpness;
- monogram strategy;
- optical balance;
- proportions;
- abstraction level;
- number of main shapes;
- color principles;
- symbol/wordmark relationship;
- small-size viability.

## Anti-Copy Rule

Use only general principles. Do not reuse a concrete silhouette, monogram linkage, composition, symbol, distinctive detail, or exact brand color combination. If a future logo resembles one specific reference, reject it and create another.
