# Evaluation Rubric

Score each concept from 1 to 10 for:

- originality;
- simplicity;
- memorability;
- brand relevance;
- similarity to reference style DNA;
- difference from specific references;
- usability at small sizes;
- one-color usability;
- SVG technical cleanliness;
- web implementation usability;
- compatibility with project design;
- favicon potential.

Select the highest total. Break ties by choosing the mark with stronger 16-24 px recognition and simpler one-color construction.

Save scores, rationale, selected concept, and assumptions to `docs/brand/concept-evaluation.md`.
